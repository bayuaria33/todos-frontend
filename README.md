## About the Project

Todos App allows you to manage your tasks efficiently by providing basic CRUD (Create, Read, Update, Delete) operations. With this API, you can easily create, retrieve, update, and delete tasks in your ToDo list application.

<br/>

## Features

- Authentication
  - (Login, Register)
- C.R.U.D
  - Create Todos
  - Read Todos
  - Update Todos
  - Delete Todos

## Installation

Clone the project

````bash
  git clone #

Go to the project directory

```bash
  cd todos-frontend
````

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
